Monotonically Ordering States in Petri Nets with Boolean Variables

This CD contains:
- a copy of the report detailing the project,
- the resulting program,
- some samples of Petri nets, and
- detailed test results

Binary is found in the PeTe-directory. It also contains the samples.

The following packages are necessary for running the program:
Qt (>= 4.7)
qmake (>= 2.0)
UnitTest++ (>= 1.4)
pkg-config (>= 0.25)
GNU Bison (>= 2.4)
flex (>= 2.5)
lp_solve (>= 5.5)
llvm (>= 2.8)
igraph (>= 0.5.3)
libproc (>= 3.2.8)
